 **新建一个项目**
	`uv init myproj`
	`cd myproj`
	`uv sync`
	`uv init` 会初始化项目，`uv sync` 会根据项目配置创建环境并同步依赖，项目里通常会出现 `pyproject.toml`、`.venv`、`uv.lock` 这些文件。
**安装依赖用：**
	`uv add fastapi uvicorn`
	`uv add pandas numpy`
**运行项目**
	`uv run uvicorn main:app --reload


uv 创建 Python 环境的常用流程基本是这几步。

**创建虚拟环境**

在项目目录下运行：

`uv venv`

默认会创建：

`.venv/`

指定 Python 版本：

`uv venv --python 3.11`

指定环境目录：

`uv venv myenv`

**激活环境**

Windows PowerShell：

`.\.venv\Scripts\activate`

CMD：

`.venv\Scripts\activate.bat`

macOS / Linux：

`source .venv/bin/activate`

激活后，命令行前面通常会出现 (.venv)。

**安装依赖**

从 requirements.txt 安装：

`uv pip install -r requirements.txt`

安装单个包：

`uv pip install requests uv pip install playwright`

查看已安装包：

`uv pip list`

导出依赖：

`uv pip freeze > requirements.txt`

**不激活也能运行**

你也可以直接用虚拟环境里的 Python：

`.\.venv\Scripts\python.exe xianyu_scraper.py "iPhone 15"`

或者用 uv run：

`uv run python xianyu_scraper.py "iPhone 15"`

**推荐的项目用法**

简单脚本项目一般这样就够了：

`uv venv .\.venv\Scripts\activate uv pip install -r requirements.txt python xianyu_scraper.py "关键词"`

如果你用的是 pyproject.toml 管理项目，可以用：

`uv init uv add requests uv add playwright uv run python main.py`

uv add 会把依赖写进 pyproject.toml，更适合长期维护的项目。
